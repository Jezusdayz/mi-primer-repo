HTML Semántico

El HTML semántico utiliza etiquetas que describen claramente el propósito del contenido. Esto mejora la accesibilidad y el SEO.

Ejemplos de etiquetas semánticas

  • <header>: Define el encabezado de una sección o página.
  • <nav>: Define un bloque de navegación.
  • <article>: Define un artículo independiente.
  • <section>: Define una sección genérica.
  • <footer>: Define el pie de página de una sección o página.
  • <main>: Define el contenido principal de la página.
No uses div para todo, elige etiquetas semánticas para mejorar la estructura y accesibilidad de tu página y SEO.

Contenedores: div vs span

<div> es un contenedor de bloque utilizado para agrupar elementos y aplicar estilos o scripts. <span> es un contenedor en línea utilizado para aplicar estilos a una parte específica del texto.
